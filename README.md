## ✨ Функционал

- 📦 Каталог товаров с загрузкой через REST API
- 🛒 Корзина — добавление, удаление, изменение количества
- ⏳ Обработка состояний загрузки и ошибок
- 📱 Адаптивный интерфейс (mobile-first)
- 🧩 SPA-архитектура на React Router
- 🔧 Компонентная структура с разделением логики и UI

## 🚀 Запуск проекта

```bash
# Клонировать репозиторий
git clone https://github.com/MeshBMW/react-shop-app.git

# Перейти в папку
cd react-shop-app

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev
```

Приложение откроется на `http://localhost:5173`
